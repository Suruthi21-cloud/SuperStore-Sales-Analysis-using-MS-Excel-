# 🛒 Superstore Sales Analysis Using SQL

## 📌 Project Overview
This project analyzes the Superstore Sales dataset using SQL to extract meaningful business insights. It includes data cleaning, exploratory data analysis (EDA), and business analysis to evaluate sales, profit, customers, and product performance.

##  SQL Concepts Used
- SELECT, WHERE, ORDER BY
- GROUP BY, HAVING
- Aggregate Functions (SUM, AVG, COUNT)
- Window Functions
- Common Table Expressions (CTEs)
- Date Functions

##  Analysis Performed
- Total Sales and Profit
- Average Order Value
- Monthly & Yearly Sales Trends
- Region & State-wise Sales Analysis
- Category & Sub-Category Analysis
- Top 10 Products and Customers
- Profit Analysis

## Conclusion
This project demonstrates practical SQL skills in data cleaning, data analysis, and business reporting. It showcases how SQL can be used to generate valuable business insights from real-world sales data.
